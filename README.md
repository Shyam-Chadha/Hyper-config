# Hyper-config
1.Download Hyper.                                                                                   
2.Select edit -> Preferences                                                                                               
3.A notepad will open.                                                                                               
4.Delete the exsisting content.                                                                                                                 
5.Paste hyper config file there.                                                                                                               
6.Save the file.                                                                                                                                    
7.Restart Hyper.                                                                                                                                
8.You are all set.
